# 🚀 HRCore Pro

## 📖 About the Project

HRCore Pro is a modern Human Resource Management System (HRMS) built with **Laravel 13**, **PHP**, **PostgreSQL**, and **Tailwind CSS**. It is designed to simplify and automate HR operations through a centralized platform with a clean, responsive, and user-friendly interface.

The system enables organizations to efficiently manage employee records, attendance, leave requests, payroll, performance, projects, tasks, reports, invoices, notifications, and internal communication, helping improve productivity and streamline daily HR workflows.

## ✨ Project Features

- 👥 Employee Management (Add, Edit, Delete & View Employees)
- 📅 Attendance Management with Status Tracking
- 🌴 Leave Management with Approval Workflow
- 💰 Payroll & Payment Management
- ⭐ Employee Performance Tracking
- 📁 Project Management
- ✅ Task Assignment & Progress Tracking
- 📄 Reports with Export & Download Options
- 🧾 Invoice Management
- 🔔 Notification System
- 💬 Internal Chat Module
- 🎂 Upcoming Birthdays & Holidays
- 📊 Interactive Dashboard with Statistics & Charts
- 🔍 Search and Filter Functionality
- 📱 Responsive & User-Friendly Interface
- 🔐 Secure Authentication & User Access
